### Primary Features

- Cross platform.
- Fast: 60 fps.
- Uses rendering engine written in C++, single view.
- Flutter framework written in dart.
- Open source.

---

### Flutter vs. Others

- Not a faceoff.
- Web tech, hybrid. (performance)
- Native wrapping. (interop bridge, map to 2 platforms)
- Full stack rendering.
